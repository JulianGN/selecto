## 📝 Descrição Detalhada (Máx. 16.000 caracteres)

```text
O Selecto é a ferramenta definitiva para engenheiros de QA, testadores e desenvolvedores frontend que desejam acelerar a escrita de testes de interface automatizados.

Com o Selecto, você não precisa mais abrir o Console do Desenvolvedor (F12) e ficar adivinhando ou construindo caminhos XPath e seletores CSS manualmente. Basta apontar, clicar e capturar!

✨ Recursos Principais:
1. Inspeção Visual Simples: Ative a ferramenta e clique em qualquer elemento da página para capturar.
2. Múltiplos Formatos Suportados:
   - CSS Selector: Seletores curtos, limpos e otimizados baseados em ID e ordem do DOM.
   - XPath: Caminhos XPath estruturados e robustos.
   - Playwright Locators: Código moderno pronto no formato page.getByRole, page.getByPlaceholder, etc.
   - Cypress: Comandos estruturados no padrão cy.get().
3. Painel Lateral Inteligente: Uma barra lateral flutuante e arrastável que fica diretamente na aba para que você não perca o foco.
4. Organizador de Elementos: Adicione rótulos e apelidos personalizados para cada elemento que você capturar.
5. Exportação em Lote: Copie todos de uma vez ou exporte sua lista de seletores em um arquivo JSON organizado para importar no seu projeto de testes.
6. Isolamento Garantido: Feito usando Shadow DOM, o que significa que o painel e os estilos do Selecto nunca quebram o layout do site que você está testando.

🚀 Como Usar:
1. Instale a extensão e clique no ícone do Selecto no navegador para abrir o painel lateral na página desejada.
2. Ative o inspetor (ou aperte o atalho rápido Alt + Shift + S).
3. Clique nos elementos HTML que deseja capturar para seus testes.
4. Adicione descrições personalizadas, mude o formato de saída a qualquer momento e clique em exportar.

Escreva testes mais rápidos para Selenium, Cypress, Playwright, Robot Framework e outros com o Selecto!
```

## 📝 Detailed Description (Max. 16,000 characters)

```text
Selecto is the ultimate Chrome extension for QA Engineers, testers, and frontend developers looking to speed up their UI test automation workflow.

With Selecto, you no longer need to open the Developer Tools (F12) and build complex XPath queries or CSS selectors manually. Just point, click, and capture!

✨ Key Features:
1. Visual Element Inspector: Toggle the inspector and click on any HTML element to capture it instantly.
2. Multiple Formats Supported:
   - CSS Selector: Generates clean, robust, and optimized CSS paths.
   - XPath: Generates reliable and structured XPath queries.
   - Playwright Locators: Ready-to-use modern selectors like page.getByRole, page.getByPlaceholder, and page.locator.
   - Cypress: Clean commands in the cy.get() syntax.
3. Floating Sidebar Panel: A draggable, collapsible sidebar that lives inside the page so you never lose focus.
4. Element Organizer: Rename and label your captured locators to organize your test suites.
5. Batch Actions: Copy all items at once or export your entire collection as an organized JSON file.
6. Fully Isolated: Built using Shadow DOM to prevent any styling issues or conflicts with the web application you are inspecting.

🚀 How to Use:
1. Open the sidebar on any website by clicking the Selecto icon.
2. Turn on the Inspector (or press the Alt + Shift + S keyboard shortcut).
3. Click on the HTML elements you want to capture for your automation scripts.
4. Rename, switch formats, and copy or export your locators.

Build faster test scripts for Selenium, Cypress, Playwright, and Robot Framework with Selecto!
```
